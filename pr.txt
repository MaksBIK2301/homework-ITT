Отчет по практике №12 "GIT" студента группы БИК2301 Соколова Максима
1) Скачиваем GIT на компьютер
2) Переходим в созданную директорию
3) Создаем в ней локальный репозиторий с расширением ".git" с помощью команды "git init"
4) Создаем текстовый файл с отчетом, добавляем файл в локальный репозиторий командой git add, сохраняем изменения с помощью git commit -m и создаем отчет в git с помощью git log
5) Добавляем новую ветку с помощью git branch и git checkout
6) Добавляем измененный отчет в основную ветку
